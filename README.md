# 🚀 Abílio Assis - Personal Website & Digital Garden

Este é o repositório do meu site pessoal e blog, desenvolvido com o framework **Astro** e inspirado pelo conceito clássico do **CSS Zen Garden**. Este projeto não é apenas um site estático, mas um laboratório de experimentação em **Programação Agêntica**.

🌐 **Live Demo:** [abiliodeassis.com.br](https://abiliodeassis.com.br)

---

## 🛠️ Stack Tecnológica

- **Framework:** [Astro](https://astro.build/) (v6.0+)
- **Tema Base:** [AstroPaper](https://github.com/satnaing/astro-paper)
- **Estilização:** Tailwind CSS
- **Linguagem:** TypeScript
- **Conteúdo:** Markdown / MDX
- **Infraestrutura:** WSL2 (Ubuntu) no Windows

---

## 🤖 Engenharia Agêntica (O Diferencial)

Este site foi construído utilizando um fluxo de trabalho de **Programação Agêntica**, onde atuei como Arquiteto e Líder Técnico, orquestrando agentes de IA para a execução:

1.  **Estratégia & Especificação:** Utilizei o **Google AI Studio (Gemini)** para refinar a arquitetura, criar User Stories e definir contratos de intenção.
2.  **Execução Agêntica:** O desenvolvimento foi realizado através do **Google Antigravity**, um agente de engenharia autônomo que operou diretamente no meu ambiente **WSL2**, realizando builds, instalações e codificação Fullstack.
3.  **Filosofia Zen Garden:** O projeto mantém a separação rigorosa entre conteúdo (HTML semântico) e apresentação (CSS), permitindo experimentações de temas variados mantendo a mesma base textual.

## 📁 Estrutura do Projeto

```text
.
├── src/
│   ├── components/     # Componentes Astro (Header, Footer, Socials)
│   ├── layouts/        # Layouts base das páginas
│   ├── data/
│   │   └── blog/       # Onde a mágica acontece (Posts em .md)
│   ├── pages/          # Rotas do site
│   └── styles/         # Configurações globais de CSS/Tailwind
├── public/             # Ativos estáticos (Imagens, Fontes)
└── astro.config.mjs    # Configurações do motor Astro
```

## 🚀 Como rodar localmente

Se você deseja explorar ou contribuir com este laboratório:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/abilioassis/personal-site.git
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o ambiente de desenvolvimento:**
   ```bash
   npm run dev
   ```
   Acesse: `http://localhost:4321`
